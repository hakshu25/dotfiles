## 主な変更点

### 1. Node.js サポート要件の変更
- Node.js 18 のサポートが終了し、**Node.js 20.19+ / 22.12+** が最低要件となった。
- これは Node.js 18 の EOL（End of Life）に合わせた対応。

### 2. デフォルトブラウザターゲットの変更
- `build.target` のデフォルト値が変更され、より現代的なブラウザをターゲットにするようになった。
- `baseline widely available` 相当の仕様に近いブラウザターゲットが基準となる方針が示された。

### 3. Rollup 4 から Rollup の最新版への追従
- バンドラーである Rollup のバージョンアップへの対応が継続して行われた。

### 4. Environment API の安定化（実験的 → 安定）
- Vite 6 で実験的に導入された **Environment API** が安定化された。
- サーバーサイド・クライアントサイドの複数実行環境を統一的に扱えるAPIで、SSR やエッジランタイムの開発をより柔軟にする。

### 5. CSS の取り扱い改善
- CSS Modules や CSS の処理パイプラインが改善された。
- Lightning CSS との連携強化。

### 6. HMR（ホットモジュールリプレースメント）の改善
- HMR の安定性とパフォーマンスが向上した。

### 7. `resolve.conditions` のデフォルト変更
- モジュール解決における条件（`conditions`）のデフォルト値が見直され、より明示的な設定が推奨されるようになった。

### 8. プラグイン API の改善
- プラグインフックに関する内部的な整理が行われ、サードパーティプラグインとの互換性が向上した。
