# Vite 7 調査結果（gemini-search スキル使用）

## 実行コマンド
```bash
gemini -p "Vite 7の主な変更点を調べて。リリース日、主要な新機能、Breaking Changes、パフォーマンス改善点など詳しく教えて" --yolo
```

## 結果

### 1. リリース日
Vite 7.0.0 は **2024年11月12日**（現地時間）に正式リリース。Vite 6のリリースから約1年後のメジャーアップデート。

### 2. 主要な新機能と改善点
- **Environment API の安定化**: Vite 6で導入されたAPIがさらに洗練され、フレームワーク開発者向けの機能が強化
- **Rollup 4 への完全準拠**: ビルドパフォーマンスと出力の最適化が向上
- **CSS 処理の改善**: デフォルトのCSSプリプロセッサの挙動やCSSモジュールのハッシュ生成アルゴリズムの安定性が向上

### 3. Breaking Changes
- **Node.js サポートの変更**: Node.js 18.20.0未満および20.10.0未満のサポートが終了
- **CJS サポートの縮小**: Vite 自体のAPIにおいてCJS形式での利用が非推奨、ESMへの移行が強く推奨
- **レガシー機能の削除**: 以前のバージョンで非推奨とされていた古い内部APIや設定オプションが完全に削除

### 4. パフォーマンス改善
- **起動時間の短縮**: ファイルスキャンと依存関係の事前バンドルプロセスが最適化
- **ビルド出力の最適化**: Rollup 4の恩恵で生成されるチャンクのサイズやTree Shakingの精度が向上
- **メモリ使用量の削減**: 開発モードにおけるモジュールグラフの管理が効率化
