# Gemini Search スキル

Gemini CLIのGoogle Search grounding機能でweb検索を行い、最新情報を取得する。

## 基本コマンド

```bash
gemini -p "<検索クエリ>" --yolo
```

- `--yolo`: ツール呼び出しを自動承認（検索に必要）
- `-m gemini-2.5-flash`: モデル指定（省略可。デフォルトで最新モデルを使用）

## 使い方

### シンプルな検索

```bash
gemini -p "調べたい内容" --yolo
```

### 詳細な検索（複数観点）

```bash
gemini -p "〇〇について、最新情報・背景・今後の動向を含めて調べて" --yolo
```

### 比較・まとめ

```bash
gemini -p "AとBを比較して。それぞれの最新バージョンと特徴を調べて" --yolo
```

## 結果の扱い方

- Geminiの出力をそのまま表示してOK
- ただし、情報の信頼性に注意。重要な情報はソースを確認するよう一言添える
- 情報の鮮度（いつ時点の情報か）が重要な場合は結果内の日付を確認する

## いつ使うか

- ユーザーが「調べて」「検索して」と明示的に言ったとき
- 最新のライブラリバージョン・APIの変更点など、知識カットオフ後の情報が必要なとき
- ニュース・時事ネタ・現在の状況を聞かれたとき
- 「〜って今どうなってる？」「最近どう？」など現在形の質問

## エラー処理

```bash
# 認証エラーの場合
gemini auth login

# CLIが見つからない場合
npm install -g @google/gemini-cli
```
